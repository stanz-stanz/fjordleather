User's reference sites: frankcleggleatherworks.com and loake.com. Both share bold confident typography on clean white backgrounds.

**Rules established:**
- `--color-stone` (#C4B5A8) is ONLY for overlines, captions, metadata labels, secondary helper text. Never for body copy, headings, or primary readable text.
- Body copy on light backgrounds must use `--color-obsidian` (#0F0D0C) or at minimum `--color-bark` (#3D2B1F) for secondary paragraphs.
- Headlines should be 56-72px+ on desktop — large and confident, not timid.
- No decorative background animations of any kind (the workshop SVG was removed).
- CTAs should use proper Button components (primary/secondary), not bare underlined links for main actions.
- Overline labels in featured sections should use cognac accent, not stone.
- Material callout descriptions should use obsidian text, not bark.

**Why:** User wants the site to feel like quiet luxury editorial (Frank Clegg, Loake) not a tech startup. High contrast, authoritative typography, product images carry the weight.
**How to apply:** On every page review, check that no stone text appears as body copy on light backgrounds. Verify headlines are bold scale. Verify no decorative animations exist.
